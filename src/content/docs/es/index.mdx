Seyfert es una maravillosa librería que se enfoca en Discord y su API para bots. Ofrece soporte para aplicaciones escalables y proporciona utilidades únicas.

:::tip
Te recomendamos tener bases sólidas en TypeScript antes de comenzar a usar Seyfert para un mejor entendimiento y fluidez.
:::

## Instalación

### Pre-requisitos

- [NodeJS](https://nodejs.org/) instalado (v18 o superior)
- [TypeScript](https://www.typescriptlang.org/) instalado (v4.9 o superior)
- NPM instalado (o un gestor de paquetes de NodeJS similar)

Seyfert también es compatible con [Deno](https://deno.land/) y [Bun](https://bun.sh/) desde la versión **2.1.0**.

### Configuración

```bash copy title="Instalando..."
npm add seyfert
```
